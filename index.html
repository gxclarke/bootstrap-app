Bootstrap App Framework

Build Responsive Web-Based Applications using Bootstrap

 

 

Get Started

 

Designed for Bootstrap 3

100% compatible with Bootstrap. Follows Bootstrap paradigms and works alongside Bootstrap to provide new capabilities useful for building web-based application user interfaces.

Powered By CSS3 Flex Box

Currently, a W3C last-call working draft. Supported by modern browsers (IE 10 or higher required). Flex Box enables fine grained control over the layout of items within a container.

User Interface Components

The Bootstrap App Framework consists of user interface components and utilities, such as advanced grid layout, off-screen panels, and navigation lists. All with full-responsive support.
